Neural
======

Python program for examining the output of neural.cpp

 * Written by: Roland Bamli
 * Function
 * - run neural.cpp
 * - save output as result.txt
 * - draw histogram from the output, save as histogram.png
 * - print outputs in ranges
 
Build
=====

>$ cmake .

>$ make

>$ python neural-hist.py
